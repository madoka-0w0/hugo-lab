# new-page — ページ生成コマンド

引数で指定されたページ名: **$ARGUMENTS**

---

まず以下を読み込んでください：
1. `.claude/skills/lab-homepage/SKILL.md`（レイアウト設計指針・コンテンツ設計）
2. `docs/requirements.md`（存在する場合 — 確定済みの技術スタック・要件）

---

## 実行手順

### 1. ページ種別の判定

`$ARGUMENTS` に指定されたページ名から、以下のいずれかを特定してください：

| ページ名キーワード | 該当ページ |
|---|---|
| トップ、ホーム、index | トップページ（index） |
| 研究、Research | 研究内容ページ |
| メンバー、Members | メンバー一覧ページ |
| 論文、Publications | 論文・研究成果ページ |
| ニュース、News | ニュース一覧ページ |
| アクセス、Contact | アクセス・連絡先ページ |
| 学生、Students | 学生向け情報ページ |
| ギャラリー、Gallery | 写真ギャラリーページ |
| その他 | 汎用ページとして設計 |

### 2. SKILL.md のレイアウト設計指針を適用

SKILL.md の「レイアウト設計」セクションに基づき、以下を守ってください：

- **ナビゲーション**: 上部固定ヘッダー（水平メニュー）
- **カラム**: 1カラム・フルワイドレイアウト
- **レスポンシブ**: スマートフォン・タブレット対応必須
- **トップページの場合**: ヒーロー画像 → 研究概要 → ニュース一覧 → 研究テーマカード → フッター の順序を守ること

### 3. ページ固有の実装指針

**トップページ（index）**:
- ヒーロー画像エリア（大型・研究室の特色を視覚的に表現）
- キャッチコピーと研究室名のオーバーレイ
- 2〜3文の研究室紹介テキスト
- ニュース最新5〜10件
- 研究テーマのカード型一覧（任意）
- フッター（連絡先・住所・所属大学）

**研究内容（Research）**:
- サブテーマごとにセクションを分割
- 各テーマに図・写真を配置
- 専門外にも伝わる概要文を冒頭に配置

**メンバー一覧（Members）**:
- 教員 → ポスドク → 博士課程 → 修士課程 → 学部生 → OB/OGの順
- 写真・氏名・所属・連絡先（任意）

**論文・研究成果（Publications）**:
- 年度別に降順で整理
- 投稿論文・学位論文・学会発表を分けて掲載
- DOIリンク付き

**ニュース（News）**:
- 時系列降順
- カテゴリタグ（受賞・論文・イベント等）があると便利

**アクセス・連絡先（Access/Contact）**:
- 地図の埋め込み（Google Maps等）
- 住所・建物名・部屋番号
- メールアドレス
- 交通アクセス案内

### 4. 技術スタックに合わせた出力

`docs/requirements.md` が存在する場合、その技術スタックに合わせてコードを生成してください。存在しない場合は、ユーザーに技術スタックを確認してください。

### 5. 出力

生成したページファイルをプロジェクトの適切な場所に配置してください。ファイル名・配置場所は技術スタックの慣習に従ってください。
